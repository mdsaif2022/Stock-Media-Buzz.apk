# Admin Panel - Add Media Guide

## How to Add Media in Admin Panel

### Step 1: Navigate to Media Management
1. Go to the Admin Panel (usually at `/admin`)
2. Click on **"Media"** in the sidebar navigation
3. You'll see the Media Management page with a list of existing media

### Step 2: Open Add Media Form
1. Click the **"Add Media"** button in the top right corner
2. The Add New Media form will appear below

### Step 3: Choose Upload Method

You have **two options** to upload media:

#### **Option A: Upload Files (Direct Upload)**
1. Make sure **"Upload Files"** tab is selected (default)
2. **Drag and drop** files onto the upload area, OR
3. **Click** the upload area to open file picker
4. Select one or multiple files from your device:
   - Images (JPG, PNG, GIF, etc.)
   - Videos (MP4, MOV, AVI, etc.)
   - Audio (MP3, WAV, etc.)
5. Selected files will appear below with file details
6. You can remove files by clicking the X button before uploading

#### **Option B: Upload via URL/Link**
1. Click the **"Upload via URL"** tab
2. Enter **Preview URL** (optional):
   - URL for thumbnail/preview image
   - Example: `https://example.com/preview.jpg`
3. Enter **File URL** (required):
   - Direct URL to the media file
   - Example: `https://example.com/video.mp4`
4. The system will download the file from the URL and upload it to your selected server

### Step 4: Fill Media Details

Fill in the required information:

1. **Title** (required): Name of the media file
2. **Category** (required): Select from dropdown:
   - Video
   - Image
   - Audio
   - Template
3. **Type/Resolution**: e.g., "4K", "1080p", "320kbps"
4. **Cloud Storage**: Select server:
   - Auto (Least Used) - automatically selects server with most space
   - Server 1
   - Server 2
   - Server 3
   - Server 4
   - Server 5
5. **Description**: Detailed description of the media
6. **Tags**: Comma-separated tags (e.g., "sunset, urban, 4k")
7. **Mark as Premium**: Checkbox to mark as premium content

### Step 5: Upload

1. Review all information
2. Click the **"Upload"** button:
   - For files: Shows "Upload X Files"
   - For URL: Shows "Upload from URL"
3. Wait for upload to complete (progress bar will show)
4. Success message will appear when done
5. Form will close automatically

### Step 6: Manage Media

After uploading, you can:
- **Search**: Use the search bar to find media
- **Filter**: Click filter button to filter by category/type
- **Edit**: Click edit icon (pencil) on any media item
- **Delete**: Click delete icon (trash) on any media item

## Tips

- **File Upload**: Best for uploading from your computer/phone
- **URL Upload**: Best for importing media from external sources
- **Multiple Files**: You can select multiple files at once when using file upload
- **Server Selection**: Use "Auto" to automatically distribute files across servers
- **Mobile Friendly**: Works on mobile devices - tap to select files

## Troubleshooting

- **Upload button disabled?**: Make sure you've selected files (for file upload) or entered File URL (for URL upload)
- **Can't see upload area?**: Make sure "Upload Files" tab is selected
- **Form not showing?**: Click the "Add Media" button again to toggle the form
